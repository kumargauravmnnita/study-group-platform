Smart Study Group Platform :

A placement preparation platform built with MERN stack.

Tech Stack :

Frontend :

- React with Vite
- HTML, CSS, JavaScript
- Socket.io Client (Real-time features)

Backend :

- Node.js
- Express.js
- MongoDB (Mongoose)
- Socket.io (WebSockets)
- ImageKit (Cloud storage)
- OpenAI API (AI features)

Features :

1. User Authentication & Profiles
2. Study Groups Management (CRUD)
3. Real-Time Group Chat
4. Resource Library with Image Upload
5. AI Question Generator
6. AI Weak Area Analyzer
7. Peer Doubt Solving Forum
8. Live Study Rooms
9. Company-Wise Prep Tracker
10. Mock Interview Scheduler
11. Leaderboards & Gamification
12. Smart Notifications System

Development Setup :

Prerequisites :

- Node.js (v20+)
- MongoDB
- npm

Installation :

Backend :

cd backend
npm install
npm run dev

Frontend :

cd frontend
npm install
npm run dev
